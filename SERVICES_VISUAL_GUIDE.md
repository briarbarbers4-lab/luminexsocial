## Customization Examples

### Change Colors
```tsx
// Main card background
background: 'rgba(11, 13, 18, 0.95)',
// Change to lighter:
background: 'rgba(247, 248, 252, 0.95)',

// Sub-card border
border: '1.5px solid rgba(13, 33, 161, 0.2)',
// Change to brighter:
border: '1.5px solid rgba(13, 33, 161, 0.5)',
```

### Change Icons
```tsx
// Replace Bot with different icon
import { Brain, Cpu, Zap } from 'lucide-react';
icon: Brain,  // Instead of Bot
```

### Change Pricing
```tsx
price: '$2,999',  // Edit this value
// Update to:
price: '$3,499',
```
