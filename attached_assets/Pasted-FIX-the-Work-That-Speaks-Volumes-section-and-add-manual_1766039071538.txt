function PortfolioSection() {

  const filteredProjects = activeFilter === 'All'

    : PORTFOLIO_PROJECTS.filter(p => p.category === activeFilter);
  
  return (
    <section className="portfolio-section">

      {/* Filters */}

          All

=== PART 4: INSTRUCTIONS FOR ADDING YOUR VIDEOS ===

STEP-BY-STEP GUIDE:

1. **Upload Your Videos:**
   - Upload to a CDN (Cloudflare, AWS S3, etc.)
   - OR place in your project's /public/videos folder
   - Get the full URL or relative path

2. **For "Work That Speaks Volumes" (5 ambient videos):**
   - Find the AMBIENT_VIDEOS object at top of file
   - Replace "YOUR_VIDEO_URL_HERE_1.mp4" with your actual video URL
   - Example:
```javascript
     const AMBIENT_VIDEOS = {
       video1: "https://cdn.luminexsocial.com/ambient/reel1.mp4",
       video2: "/videos/ambient/reel2.mp4",
       video3: "https://cdn.luminexsocial.com/ambient/reel3.mp4",
       video4: "/videos/ambient/reel4.mp4",
       video5: "https://cdn.luminexsocial.com/ambient/reel5.mp4"
     };
```

3. **For Portfolio Projects:**
   - Find the PORTFOLIO_PROJECTS array
   - For each project, update:
     * `thumbnailVideo`: Short 3-5 second preview loop
     * `fullVideo`: Complete project video
     * Update title, client, category, etc.
   - Example:
```javascript
     {
       id: 1,
       title: "Product Launch Campaign",
       client: "TechBrand Inc",
       category: "Video Editing",
       views: "2.5M",
       thumbnailVideo: "https://cdn.luminexsocial.com/thumbnails/project1-preview.mp4",
       fullVideo: "https://cdn.luminexsocial.com/projects/project1-full.mp4",
       ...
     }
```

4. **Video Requirements:**
   - Format: MP4 (H.264 codec)
   - Ambient videos: Keep under 10MB, 720p is fine
   - Thumbnail videos: 3-5 seconds, under 5MB
   - Full project videos: Can be larger, 1080p recommended
   - Compress videos before uploading

5. **Testing:**
   - Save file after updating URLs
   - Refresh page
   - Check that videos load and play
   - Test on mobile devices
